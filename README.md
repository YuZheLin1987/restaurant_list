## 功能
+ 瀏覽全部餐廳列表
+ 瀏覽單一餐廳詳細資料
+ 可使用餐廳名稱或餐廳類別來搜尋餐廳

## 開始使用
1. 請先安裝node.js以及npm
2. 將專案複製到本地
3. 使用終端機進入資料夾的位置，安裝相關套件
```
npm install
```
4. 執行程式
```
npm run start
```
5. 成功時終端機會顯示
```
This express server is running at http://localhost:3000
```
6. 到瀏覽器輸入下列網址：http://localhost:3000
7. 結束請在終端機輸入ctrl+c

## 使用工具
+ Node.js@16.17.1
+ Express@4.16.4
+ Express-handlebars@3.0.0
+ Bootstrap@5.2.1
+ FontAwesome@6.2.0
